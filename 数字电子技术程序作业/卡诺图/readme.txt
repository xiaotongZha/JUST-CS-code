1.开发环境为python 3.10.10。python3应该都可运行

2.代码主要三个部分：
gui.py:整体的ui框架搭建
anal.py:解析输入的表达式并转化为01矩阵
algo.py:遍历图找出符合要求的矩形并化简

3.源码不依赖于任何IDE,只要切换到相应目录执行命令：
python main.py

4.运行前务必看下“必看”

5.如果遇到bug或hack数据请务必联系我！！！！！！
